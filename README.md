# saber-router

> router for SPA.

```bash
# from npm
npm install saber-router

# from github
git clone https://github.com/Saber2pr/saber-router.git
```

# For Example

```ts
Router([
  {
    path: 'home',
    callback: () => console.log('home')
  },
  {
    path: 'project',
    callback: () => console.log('project')
  },
  {
    path: 'about',
    callback: () => console.log('about')
  }
])
```

---

## start

```bash
npm install
```

```bash
npm start

npm run dev

```
